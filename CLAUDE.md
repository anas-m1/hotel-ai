# hotel-ai

Production-grade Hotel Booking System built as an AI engineering learning project. Polyglot monorepo: Go microservices for core booking/inventory/pricing domain logic; Python for AI services (LangGraph agents, RAG, LiteLLM) and MCP servers. Deployed on AWS EKS across 9 phases.
